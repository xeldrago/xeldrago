How to Use This Expense Tracker

1. Four transaction types: Track Income, Expenses, Debt (money owed), and Investments.

2. Debt tracking: Use Debt type for money you owe to others or loans taken.

3. Investment tracking: Use Investment type for stocks, mutual funds, gold, etc.

4. Net Balance: Calculated as (Income - Expenses - Debt + Investment).

5. View modes: Toggle between "View Month" (shows specific month) and "View All" (shows all entries).

6. Download App: Save this app as an HTML file to use offline anywhere.

7. Export/Import: Backup or restore your financial data.

8. Undo feature: Remove the most recent entry with one click.

Note: All data is stored locally in your browser. Export regularly to backup your information.
